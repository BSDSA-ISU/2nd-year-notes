## 1. Data Types

Data types describe **what kind of data you have**.

### A. Qualitative (Categorical Data)

Describes qualities or categories. These are **labels**, not numbers used for math.

#### 1. Nominal

- No order
- Just names or categories

**Examples:**

- Blood type (A, B, AB, O)
- Eye color
- Country
- Pet type

---

#### 2. Ordinal

- Has order or ranking
- Differences between values are not measurable

**Examples:**

- Class rank (1st, 2nd, 3rd)
- Satisfaction level (Low, Medium, High)
- Shirt size (S, M, L)

---

### B. Quantitative (Numerical Data)

Numbers you can calculate with.

#### 1. Discrete

- Countable
- Whole numbers only

**Examples:**

- Number of students
- Number of books
- Goals scored

---

#### 2. Continuous

- Measured values
- Can include decimals

**Examples:**

- Height (170.5 cm)
- Weight (65.2 kg)
- Time (2.75 sec)
- Temperature

---

## 2. Measurement Scales

Measurement scales define **how meaningful numbers are** and what math operations you can do.

---

### 1. Nominal Scale

- Labels only
- No order

**Examples:**

- Gender
- Nationality
- Favorite food

---

### 2. Ordinal Scale

- Ordered categories
- Distance between values unknown

**Examples:**

- Rankings
- Grades (A, B, C)
- Survey ratings (Poor → Excellent)

---

### 3. Interval Scale

- Ordered
- Equal spacing between values
- No true zero

**Examples:**

- Temperature in Celsius or Fahrenheit
- Calendar years

Why no true zero?
> Because 0°C does NOT mean “no temperature.”

---

### 4. Ratio Scale

- Ordered
- Equal intervals
- Has true zero
- All math operations valid

**Examples:**

- Height
- Weight
- Age
- Distance
- Salary

Why true zero matters:
> 0 kg means no weight exists.

---

## 3. Quick Comparison Table

| Scale | Ordered | Equal Intervals | True Zero | Example |
| ------ | -------- | ----------------- | ---------- | -------- |
| Nominal | ❌ | ❌ | ❌ | Eye color |
| Ordinal | ✅ | ❌ | ❌ | Rank |
| Interval | ✅ | ✅ | ❌ | Celsius |
| Ratio | ✅ | ✅ | ✅ | Height |

---

## 4. Easy Memory Trick

```txt

Nominal  = Name only
Ordinal  = Order
Interval = Order + equal spacing + non true 0
Ratio    = Everything + true zero

```

## 5. Practice Examples

Identify data type **and** scale:

| Data | Type | Scale |
| ------ | ------ | ------ |
| Number of siblings | Quantitative | Ratio |
| Movie rating (1–5 stars) | Qualitative | Ordinal |
| Temperature °C | Quantitative | Interval |
| Favorite color | Qualitative | Nominal |
| Height | Quantitative | Ratio |

### Column Guide

- **Nominal** → category label (tsundere, villain, protagonist, etc.)
- **Ordinal** → ranking or placement
- **Interval** → rating scale where differences matter(1 - 10)
- **Ratio** → measurable stat with real zero (IQ, etc.)

## 7. Quick Self-Test

Try classifying these:

1. Shoe size
2. Exam score
3. Medal type (Gold/Silver/Bronze)
4. Number of pets
5. Satisfaction survey (1–10)

(Answers)

```txt

1. Quantitative – Interval or Ratio (depends on interpretation)
2. Quantitative – Ratio
3. Qualitative – Ordinal
4. Quantitative – Ratio
5. Quantitative – Interval

```

---

✅ **Summary Rule**

> If zero means "none exists" → Ratio scale  
> If values can be ranked but spacing unclear → Ordinal  
> If just names → Nominal  
> If numeric with equal spacing but no true zero → Interval
