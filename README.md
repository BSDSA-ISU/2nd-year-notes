## 📚 What even is this repo?

This repo is basically a chaotic pile of reviewer notes turned into Markdown files because:

- PDFs are annoying  
- Word files are worse  
- My attention span is illegal in 42 countries  

Markdown is clean, readable, and doesn’t try to fight me every time I press Enter. So yeah. That’s why everything is `.md`.

## 🛠 How I make these

My highly sophisticated workflow:

1. Open module  
2. Copy content  
3. Paste  
4. Pretend I understand it  
5. Add formatting so it looks like I understand it  

That’s it. No magic. No productivity guru nonsense. Just raw academic survival instinct.
