## Follow the steps below carefully

### Step 1: User Input

- Ask how many values will be entered (3–10 only)
- Use the last 3 digits of your student IDas part of your dataset
- Include:
- At least one repeated value
- At least one negative or zero value

### Step 2: Data Storage

- Store all values in an array

### Step 3: Display Values

- Displayingall entered values along with their index positions
- You are free to design your own output format

### Step 4: Data Analysis

**Compute and display the following:**

- Sum
- Average
- Highest value
- Lowest value

### Step 5: Search Feature

- Ask the user to input a number to search

### Step 6: output Results

- Display:

  - All matching index positions
  - Total occurrence count

### Step 7: Not Found Case

- If the number does not exist:
  - Display a clear message.
  - Display your own meaningful message
